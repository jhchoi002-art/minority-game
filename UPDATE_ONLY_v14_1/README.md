# 소수결 게임 Ultimate v14

교사용 화면 구조를 새로 바꾼 버전입니다.

- 교사용 화면을 한 번만 만들고 이후 필요한 부분만 갱신합니다.
- 학생 접속/제출 시 화면 흔들림과 스크롤 이동을 최소화했습니다.
- 학생 접속 현황, 팀전, 문제은행, 전자칠판, 랭킹 기능을 유지합니다.
